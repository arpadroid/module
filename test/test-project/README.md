# Test Project

This is a test project structure used for unit testing the arpadroid module builder.

## Structure

- `src/` - Source files
- `src/index.js` - Entry point
- `src/arpadroid.config.js` - Arpadroid configuration
- `src/rollup.config.mjs` - Rollup configuration
- `src/themes/` - Theme files
- `src/components/` - Component files
- `package.json` - Package configuration
- `tsconfig.json` - TypeScript configuration
